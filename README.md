# CraveBites Food Delivery System

Welcome to **CraveBites**, a food delivery application built with React (Vite) and Node.js + MongoDB.

To run the project, just open two separate terminals from the root folder and paste these exact commands.

---

## 🛠️ Start Backend (Terminal 1)

**Run this single command:**
```bash
cd backend && npm install && npm run dev
```

---

## 🛠️ Start Frontend (Terminal 2)

**Run this single command:**
```bash
cd frontend && npm install && npm run dev
```

---
